✋ Finger Counter Using OpenCV & MediaPipe
This project is a real-time finger counting application using a webcam, built with OpenCV and MediaPipe. It detects a single hand and counts the number of raised fingers based on landmark positions.

🧠 Features
Real-time finger detection

Accurate counting using hand landmarks

Simple interface with live webcam feed

Displays number of fingers raised

🛠️ Technologies Used
Python

OpenCV

MediaPipe

🚀 How It Works
Captures live video from your webcam.

Detects hand landmarks using MediaPipe.

Compares the positions of each fingertip to its lower joint to determine if it’s raised.

Displays the count on the screen in real time.
